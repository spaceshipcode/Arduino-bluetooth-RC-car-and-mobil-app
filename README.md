# Arduino-bluetooth-RC-car-and-mobil-app
A repository for building RC cars with Arduino. It also includes a mobile app with gamepad support.

⚡ Power Connections (Open Wires)

    +12V: Connect to L298N (12V Input) & Arduino (VIN)

    -12V (Common GND): Connect to L298N (GND), Arduino (GND), and all module GNDs.

🚙 L298N Motor Driver

    ENA: Arduino Pin 5

    IN1: Arduino Pin 6

    IN2: Arduino Pin 10

    ENB: Arduino Pin 3

    IN3: Arduino Pin 4

    IN4: Arduino Pin 9

    OUT1 & OUT2: Left Motor

    OUT3 & OUT4: Right Motor

📡 HC-SR04 Ultrasonic Sensor

    VCC: Arduino 5V

    GND: -12V (Common GND)

    TRIG: Arduino Pin 11

    ECHO: Arduino Pin 12

📱 Bluetooth Module (HC-05 / HC-06)

    VCC: Arduino 5V

    GND: -12V (Common GND)

    TXD: Arduino Pin 0 (RX) (⚠️ Disconnect when uploading code!)

    RXD: Arduino Pin 1 (TX) (⚠️ Disconnect when uploading code!)

🔔 Active Buzzer

    Positive (+ / Long Leg): Arduino Pin 13

    Negative (- / Short Leg): -12V (Common GND)

Arduino Bluetooth RC Araba ve Mobil Uygulama

Arduino ile uzaktan kumandalı (RC) arabalar oluşturmak için bir depo. Ayrıca gamepad destekli bir mobil uygulama içerir.
⚡ Güç Bağlantıları (Açık Uçlu Kablolar)

    +12V: L298N (12V Girişi) ve Arduino (VIN) pinine bağlayın.

    -12V (Ortak GND): L298N (GND), Arduino (GND) ve tüm modüllerin GND pinlerine bağlayın.

🚙 L298N Motor Sürücü

    ENA: Arduino Pin 5

    IN1: Arduino Pin 6

    IN2: Arduino Pin 10

    ENB: Arduino Pin 3

    IN3: Arduino Pin 4

    IN4: Arduino Pin 9

    OUT1 & OUT2: Sol Motor

    OUT3 & OUT4: Sağ Motor

📡 HC-SR04 Ultrasonik Mesafe Sensörü

    VCC: Arduino 5V

    GND: -12V (Ortak GND)

    TRIG: Arduino Pin 11

    ECHO: Arduino Pin 12

📱 Bluetooth Modülü (HC-05 / HC-06)

    VCC: Arduino 5V

    GND: -12V (Ortak GND)

    TXD: Arduino Pin 0 (RX) (⚠️ Kod yüklerken bu kabloyu çıkarın!)

    RXD: Arduino Pin 1 (TX) (⚠️ Kod yüklerken bu kabloyu çıkarın!)

🔔 Aktif Buzzer (Ses Devresi)

    Pozitif (+ / Uzun Bacak): Arduino Pin 13
    Negatif (- / Kısa Bacak): -12V (Ortak GND)
